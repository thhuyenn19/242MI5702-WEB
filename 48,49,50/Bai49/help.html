Help Page

Welcome to the help page! Below are detailed instructions to help you use the Online Classifieds Search Form effectively:

  • Search Sections: Select the section you want to search in. Options include:
    • Employment
    • For Sale
    • Housing
    • Personal
    • Miscellaneous
    • All of the above
  • Publications: Check one or more publications to narrow your search results. For example:
    • Midwest Times
    • Great Lakes Classifieds
    • Modern News
    • Employment Today
    • Middleton Daily
    • Cashtown Daily News
  • Date Range: Use the "From" and "To" date fields to filter results based on when the classifieds were posted. Make sure both fields are filled for a proper search.
  • Keywords: Enter one or more keywords to refine your search. For instance, "car", "apartment", or "job".
  • Buttons:
    • Search: Click this button to perform the search using the details you provided.
    • Help: Opens this help page for guidance.
    • Cancel: Clears all fields in the form so you can start over.

For further assistance, contact support at support@classifieds.com.
